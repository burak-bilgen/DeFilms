# DeFilms Privacy Policy

Effective date: April 24, 2026

DeFilms is a movie discovery app that uses TMDB to load movie metadata, images, videos, credits, and watch-provider information.

## Data Stored on Device

DeFilms stores the following data locally on your device:

- Local profile email and password hash in Keychain
- Local session token in Keychain
- Favorite lists in Core Data
- Recent movie searches in Core Data
- App preferences such as theme, language, and onboarding status

This data is not sent to a DeFilms server.

## Network Requests

DeFilms sends movie searches and content requests to TMDB so the app can load movie data. These requests may include the search text you type and the movie identifiers needed to load detail pages. TMDB's own privacy practices apply to TMDB services.

## Tracking and Advertising

DeFilms does not include ads, third-party analytics, or cross-app tracking.

## Account and Data Deletion

If you create a local account, you can delete it in Settings. Deleting the local account removes the local profile record, saved session, favorite lists, and recent searches associated with that profile from the device.

Guest-mode favorites and preferences remain on the device unless the app is deleted or the device data is otherwise cleared.

## Contact

For privacy questions, contact the project maintainer through the public GitHub repository.
